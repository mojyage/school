はじめに

ぷよぷよ～んとは、落ち物パズルゲーム「ぷよぷよ」の4作目です。ぷよぷよと四をかけてこのような名前になっていて、唯一対戦中に様々な効果をもたらす「特技」と対戦ルールの変わる「ステージ」というシステムが存在します。

作品紹介

プラットフォーム

PlayStation、ドリームキャスト、NINTENDO64、ゲームボーイカラー

ストーリー

初めてサーカスを見に来たアルルとカーバンクルだったが、カーバンクルが団長のマジックで消えたままサーカスが終わってしまう。そして、カーバンクルを探すための冒険がはじまる。
